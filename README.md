# Kids For Dictionary

## 기능
-해당하는 알파벳에 클릭했을 때 해당 알파벳에 대한 단어가 보임

-단어를 확인 한 뒤 더블클릭하면 해당 알파벳으로 다시 변함

### 컨셉
-귀여운 그림으로 아이들에게 친근하게 단어를 알려주는 사전
